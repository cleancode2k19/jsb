Operators:

Operators are required to do arithmetic manipulations with variable to get desired output.
  • Arithmetic Operators: Arithmetic operators(+, -, *, /, %, ++, --) are used to perform arithmetic between variables and/or values
  • Assignment Operators: Assignment Operators(=, +=, -=, *=, /=, %=) are use to assign values to js variables.
  • String Operators: String Operators(+, +=) use to add/concat strings
  • Comparison Operators: Comparison Operators(==, ===, !=, !==, >, <, >=, <=) mainly use in comparison in logical statements. For strict type and to avoid js syntax warning always use === instead of ==. As == only compare value, but === compare both value and types.
  • Conditional (Ternary) Operator: It used to present the if else condition in one line. EX: variablename = (condition) ? value1:value2
  • Logical Operators: Logical operators(&&,||, !) helps in implementation of logics.
  • Bitwise Operators: Bitwise Operators(&, |, ~, ^, <<, >>) used for operation on bit(0/1)
  • The typeof Operator: It returns type of a variable, whether itâ€™s an object, function or expression.
  • DELETE: It is use to delete object property
  • IN: in operator used to check if array index or object key is present or not. It returns value as true/false.
  • Instanceof: It used to check whether object is instance of the class/array/function.
  • Void: The void operator evaluates an expression and returns undefined. This operator is often used to obtain the undefined primitive value, using "void(0)" (useful when evaluating an expression without using the return value).
  • new : new operator use to create instance of object.
