Array:

An array contains set of item individually or as key value pair with respect to specific index.

EX:

  1. Empty array declaration in JavaScript
    var empArr = [];
  2. Array of numbers
    var numArr=[1,4,3,2,5];
  3. Array of strings
    var strArr = ['val1','val2','val3'];
  4. Assoviative array/ Array with key value pair
    var samArr = ['key':'val','key1':'val1'];

Most used array manipulation methods:

  • array.length : to identify the lenght of an array.
    var arr = [3,45,4,2,3,4]; console.log(arr.length); //output:6
  • Math.max to find the maximum value of an array item.
    var arr = [3,45,4,2,3,4]; console.log(Math.max(...arr)); //output:45
  • Math.min to find the minimum value of an array item.
    var arr = [3,45,4,2,3,4]; console.log(Math.min(...arr)); //output:2
    We have used (...) spread opeator to iterator over each element in the array other wise it will return Nan(not a number)
  • To add an item in the array push(),unshift(),splice(),concat(),(...)
    var ar = [2,1,3];
    • ar.push(66); //ar become [2,1,3,66] - added item @end of the array to ar.length position
    • ar.unshift(4); // ar become [4,2,1,3,66] - added item @ front of the array to 0 index
    • ar.splice(2,0,5,6); //ar: [4,2,5,6,1,3,66] - added 2items in the index 2 as 5,6
      2-index of array
      0-number of element to remove
      5,6- item to inject
    • var b = [41,63,28];
      ar.concat(b);//[4,2,5,6,1,3,66,41,63,28] - added an array at the end of the array
    • var x=[1,3,5],y=[2,4];
      x.splice(2,0,...b); //[1,3,2,4,5] can add array into the middle of an array
  • Various way we can remove items from an array:
    • var a = [1,2,3];
      a.pop();//[1,2] - removes item from the end of an array
    • a.shift(); // [2]- remove item from index 0
    • var ar = [1,3,5,8,9,6];
      ar.splice(2,2); //[1,3,9,6] -remove 2 elements(5,8) from index 2

Map, reduce, filter and find are 3 major array manipulation function in ECMA6.
The intend solution or apply changes to an array can be done by iteration, why we need this. When we do any itearation in js it enters into an event looping state. But these inbuilt methods given by browser processing unit, handles it differently.
Map: Performs an action over each item of an array.
var ab = [1,2,3,4,5];
 var res = ab.map(x=>2*x);//[2,4,6,8,10]
Reduce: Operation on an array items to get single output.
Ex: var a = [3,2,4];
 var res = a.reduce((x,y)=>x+y); //9
Filter: It's an operation on array to get result on specific condiction .
var a = [3,2,4,6,8];
 var res = a.filter((x)=>x>2); //[3,4,6,8]
Find: It returns first detected element in an array.
var a = [1,2,3,4,5];
 var res = a.find(x=>2); // 3

array.copyWithin(target, start, end)
Parameters:
target : The index position to copy the elements to(Required).
start : It is optional. The index position to
start copying elements from (default is 0).
end : It is optional. The index position to
stop copying elements from (default is array.length).
Return value:
It returns the modified array.

var array = [ 1, 2, 3, 4, 5, 6, 7 ];
 array.copyWithin(3); //output: [1, 2, 3, 1, 2, 3, 4]
